# Riftor
Riftor — A Simple Yet Stylish Roblox Level 3 Executor
Riftor was built with one clear goal: to run Roblox scripts easily and smoothly without unnecessary complexity. It’s a Level 3 executor, capable of executing most common scripts with stable performance — ideal for beginners or those who prefer a clean, minimal setup.

The UI is designed to be super simple and user-friendly. Everything is laid out neatly so you can launch Riftor and start using it right away — no confusing menus, no extra tweaking needed.

The Bootstrapper UI, on the other hand, gives off a completely different vibe: cool, modern, and tech-inspired, making every startup feel sleek and professional.

Overall, Riftor is the perfect choice for anyone who values simplicity, practicality, and style — lightweight, straightforward, and effortlessly cool.
